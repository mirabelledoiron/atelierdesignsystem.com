# Layout Patterns

Common page structures, grid systems, and responsive layout templates.

---

## Dashboard Layout

Sidebar + main content area — the standard for admin and app interfaces.

```tsx
<div className="flex min-h-screen">
  <aside className="w-64 border-r">Sidebar</aside>
  <main className="flex-1 p-8">Content</main>
</div>
```

## Auth Layout

Centered card for login/signup forms.

```tsx
<div className="min-h-screen flex items-center justify-center">
  <div className="w-full max-w-md">
    {/* Login/signup card */}
  </div>
</div>
```

## Split View

Two-panel layout for master-detail views (e.g., email client, file browser).

```tsx
<div className="flex h-screen">
  <div className="w-1/3 border-r overflow-y-auto">
    {/* List panel */}
  </div>
  <div className="flex-1 p-4">
    {/* Detail panel */}
  </div>
</div>
```

## Card Grid

Responsive grid of content cards — adapts from 1 to 3 columns.

```tsx
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
  {items.map(item => (
    <Card key={item.id}>
      <CardContent>{item.title}</CardContent>
    </Card>
  ))}
</div>
```

---

## When to Use Each

| Pattern | Use Case |
|---------|----------|
| Dashboard | Admin panels, analytics, settings |
| Auth | Login, signup, password reset |
| Split View | Email, messaging, file management |
| Card Grid | Product listings, portfolios, galleries |
