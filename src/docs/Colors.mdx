# Color Palette

A carefully crafted dark-first theme palette optimized for readability and visual hierarchy. All colors use HSL-based CSS custom properties via semantic tokens.

---

## Usage

```tsx
/* Use semantic tokens in your components */
<div className="bg-background text-foreground">
  <p className="text-primary">Coral accent</p>
  <p className="text-secondary">Teal accent</p>
  <p className="text-muted-foreground">Muted text</p>
</div>
```

### Installation

```sh
npm install tailwindcss-animate class-variance-authority clsx tailwind-merge
```

Copy the CSS variables from the Tokens page into your `index.css`.
