# Content States

Patterns for empty, loading, error, and success states.

---

## Empty State — No Data

When a list or page has no content yet. Include an icon, explanation, and a primary action.

```tsx
<div className="flex flex-col items-center justify-center py-16 text-center">
  <div className="w-12 h-12 rounded-full bg-muted flex items-center justify-center mb-4">
    <Inbox className="w-6 h-6 text-muted-foreground" />
  </div>
  <h3 className="text-lg font-semibold text-foreground mb-1">No projects yet</h3>
  <p className="text-sm text-muted-foreground mb-4 max-w-sm">
    Create your first project to get started.
  </p>
  <Button>Create Project</Button>
</div>
```

## Empty State — No Results

When a search or filter returns nothing. Offer a way to clear filters.

```tsx
<div className="flex flex-col items-center justify-center py-16 text-center">
  <div className="w-12 h-12 rounded-full bg-muted flex items-center justify-center mb-4">
    <Search className="w-6 h-6 text-muted-foreground" />
  </div>
  <h3 className="text-lg font-semibold text-foreground mb-1">No results found</h3>
  <p className="text-sm text-muted-foreground mb-4 max-w-sm">
    Try adjusting your search or filter criteria.
  </p>
  <Button variant="outline">Clear Filters</Button>
</div>
```

## Loading State — Skeletons

Skeleton placeholders while data loads. Mirror the layout of the content being loaded.

```tsx
<div className="space-y-4 max-w-md">
  <div className="flex items-center gap-3">
    <Skeleton className="w-10 h-10 rounded-full" />
    <div className="space-y-2 flex-1">
      <Skeleton className="h-4 w-3/4" />
      <Skeleton className="h-3 w-1/2" />
    </div>
  </div>
  <Skeleton className="h-24 w-full rounded-xl" />
</div>
```

## Loading State — Spinner

Inline spinner for buttons or section loading.

```tsx
{/* Button spinner */}
<Button disabled>
  <div className="w-4 h-4 border-2 border-primary-foreground/30 border-t-primary-foreground rounded-full animate-spin mr-2" />
  Loading...
</Button>

{/* Inline spinner */}
<div className="flex items-center gap-2 text-sm text-muted-foreground">
  <div className="w-4 h-4 border-2 border-primary/30 border-t-primary rounded-full animate-spin" />
  Fetching data...
</div>
```

## Error State

When something goes wrong. Be specific and offer recovery actions.

```tsx
<div className="flex flex-col items-center justify-center py-16 text-center">
  <div className="w-12 h-12 rounded-full bg-destructive/10 flex items-center justify-center mb-4">
    <AlertTriangle className="w-6 h-6 text-destructive" />
  </div>
  <h3 className="text-lg font-semibold text-foreground mb-1">Something went wrong</h3>
  <p className="text-sm text-muted-foreground mb-4 max-w-sm">
    We couldn't load this page. Please try again or contact support.
  </p>
  <div className="flex gap-3">
    <Button>Try Again</Button>
    <Button variant="outline">Contact Support</Button>
  </div>
</div>
```

## 404 State

Page not found — keep it simple with a way to navigate home.

```tsx
<div className="flex flex-col items-center justify-center py-16 text-center">
  <div className="w-12 h-12 rounded-full bg-muted flex items-center justify-center mb-4">
    <FileX className="w-6 h-6 text-muted-foreground" />
  </div>
  <h3 className="text-4xl font-bold text-foreground mb-2">404</h3>
  <p className="text-sm text-muted-foreground mb-4">This page doesn't exist or has been moved.</p>
  <Button variant="outline">Go Home</Button>
</div>
```

## Success State

Confirmation after a completed action.

```tsx
<div className="flex flex-col items-center justify-center py-16 text-center">
  <div className="w-12 h-12 rounded-full bg-secondary/10 flex items-center justify-center mb-4">
    <CheckCircle className="w-6 h-6 text-secondary" />
  </div>
  <h3 className="text-lg font-semibold text-foreground mb-1">Payment successful</h3>
  <p className="text-sm text-muted-foreground mb-4 max-w-sm">
    Your payment of $49.00 has been processed. A receipt has been sent to your email.
  </p>
  <Button>View Receipt</Button>
</div>
```

---

## State Checklist

| State | Icon | Background | Action |
|-------|------|-----------|--------|
| Empty (no data) | `Inbox` | `bg-muted` | Primary CTA |
| Empty (no results) | `Search` | `bg-muted` | Clear Filters |
| Loading | Skeleton / Spinner | — | — |
| Error | `AlertTriangle` | `bg-destructive/10` | Try Again + Support |
| 404 | `FileX` | `bg-muted` | Go Home |
| Success | `CheckCircle` | `bg-secondary/10` | View / Continue |
