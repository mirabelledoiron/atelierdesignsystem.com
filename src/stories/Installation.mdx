# Installation

To use Atelier primitives in your project, follow these steps.

### 1. Install Dependencies

Atelier UI primitives rely on Tailwind + animation utilities and Lucide icons.

```bash
npm install lucide-react tailwindcss-animate
```

### 2. Configure Tailwind

Atelier relies on **HSL tokens** exposed as CSS variables (e.g. `--primary`, `--background`) and uses `--radius` for the base radius scale.

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: "hsl(var(--primary))",
        background: "hsl(var(--background))",
      },
      borderRadius: {
        lg: "var(--radius)",
        md: "calc(var(--radius) - 2px)",
      },
    },
  },
};
```

### 3. Import Styles

Make sure your app loads the global CSS that defines tokens and fonts (e.g. in `main.tsx` / `_app.tsx`).

```javascript
import "./index.css";
```
