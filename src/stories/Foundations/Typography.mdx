# Typography

Atelier uses **Inter** for UI elements and **JetBrains Mono** for code and technical data.

export const typography = {
  type: {
    primary: '"Inter", system-ui, sans-serif',
    code: '"JetBrains Mono", monospace',
  },
  weight: {
    regular: "400",
    medium: "500",
    semibold: "600",
    bold: "700",
  },
  size: {
    s1: 12,
    s2: 14,
    s3: 16,
    m1: 20,
    m2: 24,
    m3: 28,
    l1: 32,
    l2: 40,
    l3: 48,
  },
};

export const SampleText = "The quick brown fox jumps over the lazy dog.";

## Inter (Sans-serif)

**Weight: 400, 500, 600, 700**
